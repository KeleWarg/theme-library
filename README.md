# Kele's Design System

Multi-theme design tokens for AI-assisted development.

## Install

```bash
npm install @kele/design-system
```

## Usage

```jsx
import '@kele/design-system/css';

<html className="theme-health-sem">
  <button style={{ backgroundColor: 'var(--color-btn-primary-bg)' }}>
    Click me
  </button>
</html>
```

## Themes

- `theme-health-sem`
- `theme-home-sem`
- `theme-forbes-media-seo`
- `theme-advisor-sem-compare-coverage`
- `theme-llm`
